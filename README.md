# Chat
Урок 5. Помогаем клубу анонимных геймеров.  
Курс Асинхронный Python от Devman.

## Запуск
Установите зависимости
```bash
$ pip install -r requirements.txt
```

Настройте скрипт через переменные окружения или аргументы CLI
```bash
$ export SERVER_HOST="" \
  LISTEN_PORT=0 \
  SEND_PORT=0 \
  LOG_PATH="" \
  USER_HASH="" \
```

Запустите проект
```bash
$ python chat.py
```
